# Superfrej User Guide

Welcome! Superfrej helps you run your short-term rentals: properties, calendar, reservations, guests, payments and channel sync in one place.

## Chapters

Chapters appear here as each module ships:

1. [Managing your rentals](rentals.md)
2. [Recording reservations](reservations.md)
3. [Using the calendar](calendar.md)
4. [Setting your prices](pricing.md)
5. [Your direct booking site](booking-site.md)

## Running the app

1. Install [Node.js](https://nodejs.org) 20 or newer.
2. In the project folder run `npm install`, then `npm run dev`.
3. Open [http://localhost:3000](http://localhost:3000).

Your data is stored locally in `data/superfrej.db` — back up that single file to back up everything.
